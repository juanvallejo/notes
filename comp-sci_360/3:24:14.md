**Python**

- Author (The Netherlands 1989)

- Name

    - Free
    - Object oriented
    - Portable
    - Dynamically typed
    - Automatic memory management
    - Library utilities

        - Expression: 1 < 2

    - CHaracters

    - Dictionaries
        - {'food':'spam','taste':'yum'}
    - Other
        - Tuples
